Digital IC Trainer Kit Version 1.2
A hybrid Hardware-Software testing environment designed to interface an Arduino Uno with a custom-built BCD to 7-Segment Decoder circuit. This kit allows for manual bit-entry via Serial Monitor and automated logic verification (Auto-Scan) for discrete logic gate troubleshooting.

🚀 Features
4-Bit BCD Control: Send binary strings (e.g., 0001, 1100) from your PC to drive physical logic gates.

Real-time Status Monitoring: High-speed feedback for S (Select), E (Enable), and F (Feedback) pins.

Auto-Test Mode: A built-in diagnostic tool that cycles through all 16 binary states (0-15) to verify the Truth Table of the connected ICs.

Logic Debugging: Specifically designed to identify "invalid" BCD states (10-15) and hardware "blanking" behaviors.
